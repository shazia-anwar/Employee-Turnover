# Employee Turnover Prediction

## Project Overview
This project predicts whether an employee is likely to leave the organization based on HR, performance, and compensation-related features.

The objective is to help organizations identify employees at risk of turnover and support data-driven retention strategies.

---

## Dataset Description
The dataset includes employee-related attributes such as:

- Job_Satisfaction
- Performance_Rating
- Years_At_Company
- Work_Life_Balance
- Distance_From_Home
- Monthly_Income
- Education_Level
- Age
- Num_Companies_Worked
- Employee_Role
- Department
- Annual_Bonus
- Training_Hours
- Annual_Bonus_Squared (engineered feature)
- Annual_Bonus_Training_Hours_Interaction (engineered feature)
- Employee_Turnover (Target Variable)
